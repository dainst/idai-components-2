# Configuraton.json

## Top level elements 

The top level elements are

```
{
  "types": [],
  "relations" : []
}
```

## Type definitions

Types are defined as shown here

```
{
  "type": "a_type_identifier",
  "label" : "A label",
  "description" : "a text shown on hovering over the type label"
  "fields": []
}
```

## Field definitions

Fields look like

```
{
  "name" : "the_field_identifier",
  "label" : "The field label"
}
```
